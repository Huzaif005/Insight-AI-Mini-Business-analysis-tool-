# InsightAI Workflow

This document describes the high-level workflow for the InsightAI project.

1. Upload dataset via frontend upload UI.
2. Backend saves file to `uploads/`.
3. Data cleaning and preprocessing run in `services/data_cleaning.py`.
4. Visualization metadata generated by `services/visualization_service.py`.
5. Predictions via `services/prediction_service.py`.
6. Insights created by `services/insight_generator.py`.
7. Reports generated and saved in `reports/`.
